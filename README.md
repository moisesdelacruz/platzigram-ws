# Platzigram Realtime Backend, created with [Platzi][platzi] in the [professional course Nodejs][course].

## Install:
```sh
  $ git clone https://github.com/moisesdelacruz/platzigram-ws.git

  $ cd platzigram-ws/

  $ npm install

  $ npm start
```
